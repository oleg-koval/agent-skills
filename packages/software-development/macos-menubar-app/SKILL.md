# Production macOS Menubar App with SwiftUI

## Overview

Build a fullscreen or menubar-resident macOS app in SwiftUI with:
- MenuBarExtra for always-visible menubar item
- Sandbox entitlements for App Store submission
- AVFoundation audio integration
- Global keyboard shortcuts (Command Registry)
- Settings panel and onboarding UI
- App Store submission readiness

## Architecture

```
MyApp.swift                    # App entry point + @main
├── MainView                    # Window/menubar UI
├── Settings                    # Preferences panel
├── AudioManager               # AVFoundation wrapper
├── ShortcutMonitor            # Global keyboard shortcuts
└── AppDelegate (if needed)    # Lifecycle hooks
```

## Step 1: Switch to SwiftUI (if migrating from UIKit)

If starting fresh with SwiftUI:

```swift
@main
struct MyApp: App {
    @NSApplicationDelegateAdaptor(AppDelegate.self) var delegate
    
    var body: some Scene {
        MenuBarExtra("MyApp", systemImage: "app.fill") {
            ContentView()
                .frame(width: 350, height: 500)
        }
        .menuBarExtraStyle(.window)
        
        Settings {
            SettingsView()
        }
    }
}
```

## Step 2: Configure Sandbox Entitlements

For App Store submission, add to `MyApp.entitlements`:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <!-- Basic sandbox -->
    <key>com.apple.security.app-sandbox</key>
    <true/>
    
    <!-- Audio capabilities -->
    <key>com.apple.security.device.audio-input</key>
    <true/>
    
    <!-- User-selected files -->
    <key>com.apple.security.files.user-selected.read-write</key>
    <true/>
    
    <!-- Network (if needed) -->
    <key>com.apple.security.network.client</key>
    <true/>
    
    <!-- Disable unrestricted network (avoid App Review rejection) -->
    <!-- Avoid broad file system access -->
    
</dict>
</plist>
```

Add entitlements file to Xcode:
1. Build Settings → Code Signing Entitlements → set to `MyApp.entitlements`
2. Verify in Build Phases → Copy Bundle Resources

## Step 3: AVFoundation Audio Integration

Create an `AudioManager` for sound effects:

```swift
import AVFoundation

class AudioManager {
    static let shared = AudioManager()
    
    private var players: [String: AVAudioPlayer] = [:]
    
    func playSound(_ name: String, volume: Float = 1.0) {
        guard let url = Bundle.main.url(forResource: name, withExtension: "wav") else {
            print("Sound not found: \(name)")
            return
        }
        
        do {
            let player = try AVAudioPlayer(contentsOf: url)
            player.volume = volume
            player.play()
            
            // Keep reference to prevent deallocation
            players[name] = player
            
            // Clean up after playback
            DispatchQueue.main.asyncAfter(deadline: .now() + player.duration + 0.1) {
                self.players[name] = nil
            }
        } catch {
            print("Error playing sound: \(error)")
        }
    }
}
```

Usage:

```swift
Button("Play Sound") {
    AudioManager.shared.playSound("notification")
}
```

## Step 4: Global Keyboard Shortcuts

Use Command Registry (macOS 14+) for global shortcuts:

```swift
import SwiftUI

struct MyApp: App {
    @State private var isActive = false
    
    var body: some Scene {
        MenuBarExtra("MyApp", systemImage: "app.fill") {
            ContentView(isActive: $isActive)
        }
        .keyboardShortcut(KeyEquivalent("n"), modifiers: .command)
        .onChange(of: isActive) { oldValue, newValue in
            if newValue {
                AudioManager.shared.playSound("activated")
            }
        }
    }
}
```

For pre-macOS 14, use `Carbon` event monitoring (less reliable):

```swift
import Carbon

class ShortcutMonitor {
    var eventHandler: EventHandlerRef?
    
    func startMonitoring(modifiers: UInt32, key: UInt32, callback: @escaping () -> Void) {
        let eventSpec = EventTypeSpec(
            eventClass: OSType(kEventClassKeyboard),
            eventKind: UInt32(kEventHotKeyPressed)
        )
        
        InstallApplicationEventHandler(
            { (next, inEvent, userData) -> OSStatus in
                callback()
                return noErr
            },
            1,
            &eventSpec,
            nil,
            &eventHandler
        )
    }
}
```

## Step 5: App Menu & Settings

Ensure standard macOS app menu with Quit ⌘Q:

```swift
@main
struct MyApp: App {
    @NSApplicationDelegateAdaptor(AppDelegate.self) var delegate
    
    var body: some Scene {
        MenuBarExtra("MyApp", systemImage: "app.fill") {
            ContentView()
        }
        .menuBarExtraStyle(.window)
        
        Settings {
            SettingsView()
        }
    }
}

class AppDelegate: NSObject, NSApplicationDelegate {
    func applicationDidFinishLaunching(_ notification: Notification) {
        installMainMenu()
    }
    
    func installMainMenu() {
        let mainMenu = NSMenu()
        let appMenu = NSMenu()
        
        appMenu.addItem(NSMenuItem(
            title: "Settings...",
            action: #selector(NSApplication.unhide(_:)),
            keyEquivalent: ","
        ))
        
        appMenu.addItem(NSMenuItem.separator())
        
        appMenu.addItem(NSMenuItem(
            title: "Quit MyApp",
            action: #selector(NSApplication.terminate(_:)),
            keyEquivalent: "q"
        ))
        
        mainMenu.addItem(NSMenuItem(title: "MyApp", submenu: appMenu))
        NSApplication.shared.mainMenu = mainMenu
    }
}
```

## Step 7: App Store Submission Checklist

Before submitting:

### Code
- [ ] MenuBarExtra configured
- [ ] App menu with Quit ⌘Q (always instantiate)
- [ ] Keyboard shortcuts work
- [ ] Audio plays on test device
- [ ] No console errors
- [ ] Settings panel functional

### Metadata
- [ ] App name is specific (not generic like "Mac App")
- [ ] Subtitle is descriptive
- [ ] Description lists features, not just "an app"
- [ ] Keywords accurate
- [ ] Screenshots show actual UI (not mock-ups)
- [ ] Privacy policy URL provided

### Entitlements
- [ ] Only required entitlements (no unrestricted network)
- [ ] Sandbox enabled
- [ ] Audio entitlement if using AVFoundation
- [ ] File access only if actually needed

### Build
- [ ] Increment CFBundleVersion
- [ ] macOS deployment target = 11.0 or higher
- [ ] Code signed correctly
- [ ] Notarization enabled (if distributing outside App Store)

## Common Issues & Fixes

| Issue | Cause | Fix |
|-------|-------|-----|
| Audio not playing | AudioPlayer deallocated | Keep reference in `players` dict |
| Keyboard shortcut not working | Global shortcut requires user grant | Check System Settings → Keyboard Shortcuts |
| MenuBarExtra not visible | Window style wrong | Use `.menuBarExtraStyle(.window)` |
| App Review: Quit unavailable | Quit only in one code path | Always instantiate app menu in AppDelegate |
| Entitlement rejection | Unnecessary sandbox entitlements | Remove unused capabilities |
